{# ── Step-by-step guide ──────────────────────── #}
1
Remove Duplicates
A host can appear many times in the CSV. This keeps only the latest entry per host.
2
Show Disconnected Only
Removes hosts that are currently online, leaving only the ones that need attention.
3
Ping Hosts
Sends a network ping to each host to check if the machine is reachable. This tells you if the host is down or just the Cribl agent.
4
Look Up in ServiceNow
Checks each host in the CMDB to find decommissioned or retired machines that can be safely removed from Cribl.
5
Diagnose
Labels each host: "Agent Down" (host is up but Cribl isn't), "Host Down", or "Decommissioned".
6
Download Report
Downloads everything as a formatted Excel spreadsheet you can share or work with.
Start Over
Undo all changes and go back to the original uploaded data.
