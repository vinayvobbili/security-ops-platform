Visual Insights i What is Visual Insights?

The dashboard charts and tables show counts — top actors, top techniques, etc. Visual Insights shows the connections between those entities: which actors share techniques, how malware links actors to regions, and how intelligence flows from priority to action.

All diagrams respect the date filter at the top, support dark mode, and can be exported as SVG or PNG for reports.

Relationship diagrams across actors, techniques, malware, and intelligence priorities

Top 8 threat actors mapped to their most-used MITRE techniques. Techniques color-coded by detection rule coverage: green = 5+ rules, yellow = 1-4 rules, red = 0 rules. i How to read this diagram

Actors (blue, left) connect to their most-used MITRE techniques (right). When two actors point to the same technique, that's a convergence point — one detection rule covers multiple actors.

Color coding comes from your Detection Rules page:
• Green = 5+ detection rules (strong coverage)
• Yellow = 1-4 rules (moderate coverage)
• Red = 0 rules (detection gap)

Red nodes are your highest-priority gaps — techniques actively used by threat actors with no detection rules in place.

Intelligence flow from priority levels through action types to top threat actors. Shows how critical intel maps to specific actors. i How to read this diagram

Three columns flow left to right:
• Priority levels (purple) — Critical, High, Medium, Low, Info
• Action types (cyan) — Detection Opportunity, Hunt Opportunity, None Required
• Top actors (blue) — the 8 most-mentioned threat actors

Look for patterns: are Critical tippers mostly flowing to "None Required"? That's a gap worth investigating. Counts on each node are accurate totals from the dashboard data.

Three columns flow left to right:
