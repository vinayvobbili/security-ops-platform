Onboarding snippets

Copy-paste these to onboard a new user to the public MCP endpoint. One PAT (minted at /account) works for both the local-LLM CCR shim and the public MCP endpoint. The first three variants reuse the ANTHROPIC_AUTH_TOKEN the user has already exported for Claude Code, so they don't have to paste the token a second time — pick the one that matches their shell. Fall back to the literal-PAT variant if they haven't set up Claude Code yet.

macOS / Linux — bash / zsh
claude mcp add --transport http --scope user ir-mcp https://gdnr.the-company.com/mcp --header "Authorization: Bearer $ANTHROPIC_AUTH_TOKEN"
Windows — PowerShell
claude mcp add --transport http --scope user ir-mcp https://gdnr.the-company.com/mcp --header "Authorization: Bearer $env:ANTHROPIC_AUTH_TOKEN"
Windows — CMD
claude mcp add --transport http --scope user ir-mcp https://gdnr.the-company.com/mcp --header "Authorization: Bearer %ANTHROPIC_AUTH_TOKEN%"
Literal PAT (fallback — if Claude Code env vars aren't set)
claude mcp add --transport http --scope user ir-mcp https://gdnr.the-company.com/mcp --header "Authorization: Bearer <your-PAT>"
