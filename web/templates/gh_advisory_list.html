How Do Cyber Security Advisories Work?

Critical security advisories affecting our stack — pulled hourly by the Detection Engineering poller from GitHub (reviewed + malware), OSV (malicious packages), CISA KEV, and any feeds you add under Sources. Review each one, record your assessment, then either close it as not worth reporting or escalate it to the Package Compromise Assessment Teams channel — duplicate-safe.

🔎 Look up a CVE or GHSA that isn't in the queue
🔎

GitHub can take weeks to publish a GHSA after a CVE is assigned. This fetches it on demand from GitHub → NVD → OSV and adds it to the queue.

🎯

What it's for

Several feeds publish security advisories for open-source packages and exploited vulnerabilities. This queue surfaces the critical ones relevant to us so a human decides whether each warrants escalation — instead of them scrolling past in a feed.

📡

1. Hourly poll, many sources

A Detection Engineering job pulls new advisories every hour from GitHub (reviewed-critical + malware), OSV (malicious npm/PyPI packages), CISA KEV (known-exploited CVEs), and any RSS feeds or extra OSV ecosystems added under Sources. Each lands as New. The same finding reported by more than one feed is de-duplicated. A feed's pre-existing backlog is baselined silently and kept hidden, so the page only ever shows genuinely new work.

📝

2. Review & record

Open an advisory to see severity, CVSS, CWEs, affected packages, and the full description. Capture your assessment in the triage notes — what it affects, our exposure, and the rationale for the call. The chat bubble can answer questions about anything on the page.

🚦

3. Decide

Either Close — not worth reporting (with your reasoning preserved), or Notify the Package Compromise Assessment Teams channel. Escalation is duplicate-safe: an advisory already reported won't fire a second message.

⚙️

Sources

Use Sources to switch feeds on/off, add an RSS/blog feed (each post is screened by AI for package-compromise relevance) or an extra OSV ecosystem (Go, Maven, …). Need a source that isn't a feed? Request it and the team gets pinged to wire it up.
