📋 Expected format

Any of these work — we'll auto-detect the IP column:

  • One column of IPs (header row optional):  ip → 8.8.8.8
  • Multiple columns — we'll pick the one with the most IP-like values:  timestamp,src_ip,dst_ip,user
  • Extras like 1.2.3.4:443 (port) or 1.2.3.4/24 (CIDR) are tolerated.
  • Both IPv4 (8.8.8.8) and IPv6 (2001:db8::1) are supported.
